2. Principes de protection des données

ALB Immobilier s'engage à respecter le Règlement Général sur la Protection des Données (RGPD) et la loi française "Informatique et Libertés".

Principes fondamentaux :

  • Confidentialité par défaut : Tous les données personnelles sont strictement confidentielles
  • Transparence : Vous savez exactement quelles données nous collectons et pourquoi
  • Contrôle : Seule Joce peut autoriser la révélation de données personnelles à un tiers
  • Sécurité : Mesures techniques et organisationnelles pour protéger vos données

3. Données collectées

Pour tous les utilisateurs :

  • Email
  • Prénom et nom
  • Numéro de téléphone (facultatif)
  • Ville/code postal
  • Données de connexion (timestamp, adresse IP anonymisée)

Pour les professionnels :

  • Zones d'intervention
  • Description professionnelle
  • Photo/avatar
  • SIRET/numéro d'inscription (jamais publié, stocké de manière sécurisée)
  • Statut de disponibilité

Pour les annonces immobilières :

  • Ville et code postal (aucune adresse précise)
  • Type de bien (maison/appartement)
  • Description du bien
  • Photos/vidéos du bien
  • Prix (si applicable)

4. Base légale du traitement

  • Consentement explicite : Vous acceptez nos conditions et politique lors de l'inscription
  • Exécution du service : Nécessaire pour fonctionner la plateforme
  • Obligations légales : Archivage, traçabilité, lutte anti-fraude

5. Publicité des données

Règle stricte : Les données personnelles NE SONT JAMAIS AUTOMATIQUEMENT PUBLIQUES

Données JAMAIS publiées :

  • Numéro de téléphone
  • Email complet
  • SIRET / numéro d'inscription
  • Adresse complète
  • Adresse précise des biens

Données PARFOIS visibles (version publique réduite) :

  • Prénom et nom (uniquement pour les pros validés)
  • Ville et code postal (pas l'adresse exacte)
  • Photo de professionnel
  • Zone d'intervention générale
  • Description professionnelle
  • Note et nombre d'avis

⚠️ Révélation de données personnelles : Seule Joce peut autoriser la révélation de données personnelles (email, téléphone, adresse exacte) à un tiers. Jamais automatique, jamais par défaut. Le particulier ou le pro doit explicitement consentir à partager ses données.

6. Durée de conservation

  • Comptes actifs : Conservés aussi longtemps que le compte est actif
  • Comptes inactifs : Suppression après 2 ans d'inactivité
  • Données de transaction : Conservées 3 ans (obligation légale)
  • Logs techniques : 6 mois
  • Demande de suppression : Traitée dans les 30 jours

7. Partage des données

Vos données ne sont partagées avec aucun tiers sauf :

  • Supabase : Hébergement de base de données (conformité RGPD via contrats de traitement de données)
  • Brevo : Envoi de mails transactionnels (CGU de Brevo validée)
  • Vercel : Hébergement frontend (conformité RGPD)
  • Obligations légales : Autorités si requête judiciaire valide

8. Transferts internationaux

Toutes les données sont hébergées en Europe (Supabase, Vercel, Brevo tous certifiés RGPD). Aucun transfert vers pays tiers sans accord préalable.

9. Vos droits

Vous avez le droit à :

  • Droit d'accès : Demander une copie de vos données
  • Droit de rectification : Corriger des données inexactes
  • Droit à l'oubli : Demander la suppression de vos données
  • Droit à la portabilité : Récupérer vos données dans un format standard
  • Droit d'opposition : S'opposer au traitement de vos données
  • Droit de limitation : Limiter le traitement temporaire

Pour exercer l'un de ces droits, contactez : contact@albimmobilier.fr

10. Cookies et suivi

  • Cookies de session : Essentiels au fonctionnement (token d'auth)
  • Cookies analytiques : Aucun tiers (pas de Google Analytics, etc.)
  • Tracking : Pas de suivi comportemental

11. Sécurité des données

Mesures mise en place :

  • Chiffrement TLS/HTTPS pour tous les transferts
  • Bases de données protégées par authentification forte
  • Tokens d'API stockés de manière sécurisée (Supabase Vault)
  • Pas de stockage de mots de passe en clair
  • Accès à données personnelles limité à Joce

12. Incident de sécurité

En cas de fuite ou compromission de données, les utilisateurs affectés seront notifiés dans les 72 heures avec :

  • Nature de la fuite
  • Données concernées
  • Actions correctives
  • Contact pour plus d'info

13. Sous-traitants (DPA)

Liste des sous-traitants conformes au RGPD :

  • Supabase : Hébergement données (Europe)
  • Brevo : Emails transactionnels (Europe)
  • Vercel : Hébergement frontend (Global Edge Network, compliant)
  • Zimbra/OVH : Emails professionnels (France)

Tous ces prestataires ont signé des contrats de traitement de données conformes au RGPD (DPA - Data Processing Agreements).

14. Modifications de cette politique

ALB Immobilier se réserve le droit de modifier cette politique à tout moment. Les modifications seront publiées sur cette page avec une date de mise à jour. L'utilisation continue après modification signifie acceptation des changements.
